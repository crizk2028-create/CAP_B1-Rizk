Future

The skills I have learned in this class include how to use App Lab, Game Lab, and Web Lab in Code.org. In App Lab, I have learned how to design and make apps using algorithms, functions, and if else statements. I learned how to store variables, vanish items, switch screens, and create scenes using a turtle. In Game Lab I learned how to create and move sprites, how to use a for loop and draw loop functions, and how to make an interactive game where the user can control sprites and win or lose. While my future career does not entail any computer science related skills, I would like to continue creating apps, websites, and games in my free time and expand my skills in this area. In this class, I have learned that I can catch onto coding pretty quickly, and I have learned how to go slow, and critically think when I am going through my code trying to find any bugs or mistakes. Some areas that are still a weakness for me are the pictures in Web Lab. I am still a bit hesitant and it takes me a while when I try to reposition the images to the places I want them, and to continue to grow in this area, I plan to practice more and keep trying until it becomes like second nature. After becoming a master in the basics, I can continue to expand my horizons until I have learned advanced ways to make a website, organize pictures or text boxes, and add color and personality to a website. This class has taught me how to use the basics and build off of them until you can create something truly beautiful, and complex.
