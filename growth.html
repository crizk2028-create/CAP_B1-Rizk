Growth

Unit 3 Game Lab Project

This work represents an area of growth because it demonstrated my expanded knowledge on gamelab. When I first started learning how to code on Game Lab, I had to get used to all the new codes and differences from App Lab, and this project shows how much ime I put into learning the new codes, and creating them into a fun game about saving a spaceship from an evil alien. The hardest area for me to master was the disappearing of the stars because I couldn’t figure out how to make the stars disappear after the alien bounced off once. This was the hardest because while I was able to make them vanish, it took me a while to figure out how to make the alien pass through the stars after they had turned invisible. This was a major obstacle I had to get past and by experimenting with the code and asking friends I was able to figure it out and finish this project. I put in a lot of time and effort into this project to make it look good and function properly, as I do with all of my projects for this class, and I believe it directly influenced my success by making these projects work without any bugs. It also made me a better coder because instead of just looking up the answers to the codes, I would use trial and error to see what worked and what didn’t, and I have learned from those efforts. Knowing what I know now at the end of this year, I would say there is nothing that needs to be changed, but if I had more time on this project I would have displayed the score on the screen so the player can see how many points they got and how many lives they have left.
