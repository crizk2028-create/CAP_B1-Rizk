Best Work

Unit 6 Turtle Programming Project-B1

This project is an example of my best work because it shows the effort I spent to make a scene that looks pretty and includes some fluctuating objects like the bird that changes color depending on the amount of fish drawn, which was randomized. I am very proud of the final product and the sunset, and in my opinion the way the sun looks with its rays makes it really unique. This work was not necessarily hard to complete, but it took a lot of time. The pieces on their own were easy, like the bubbles and the foam for the waterfall, but putting it all together and making it run in order was a bit complicated and took some time to figure out. To create this work, I used my knowledge of functions, function draw if else statements, and for loops.
